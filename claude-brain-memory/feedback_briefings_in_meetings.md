Briefing notes always go in `meetings/`, never a separate `briefings/` folder.

**Why:** User found the split messy — some briefings were landing in `briefings/` and some in `meetings/`. One location is cleaner.

**How to apply:** When creating a pre-meeting briefing note, save it as `meetings/YYYY-MM-DD-<subject>-briefing.md` (with `-briefing` suffix to distinguish from the actual meeting note). Never create a `briefings/` directory.
